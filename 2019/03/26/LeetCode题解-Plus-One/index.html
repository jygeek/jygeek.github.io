LeetCode题解-Plus One

发表于 2019-03-26 | 分类于 算法

分析

这题主要是一个进位问题，给出一个数以数组形式存放，然后将这个数加一，但不包括以0开头的数，除了0本身。

Example 1:
Input: [1,2,3]
Output: [1,2,4]
这表示123及其加一后的124

1.思路

一个数加一有三种情况 ：

a.当它的最低位小于9时，这时直接在最低位加一即可

b.当它的最低位等于9时，此时令最低位为0，在最低位的上一位加1即可

c.一种特殊情况，即这个数为9999….时
